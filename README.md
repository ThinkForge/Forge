# Usage
<!-- usage -->
```sh-session
$ npm install -g @thinkforge/forge
$ forge COMMAND
running command...
$ forge (-v|--version|version)
@thinkforge/forge/0.0.1-ALPHA win32-x64 node-v9.11.1
$ forge --help [COMMAND]
USAGE
  $ forge COMMAND
...
```
